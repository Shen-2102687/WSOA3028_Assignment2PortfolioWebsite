Ninja vs Sushi is a game inspired by the original game Angry Birds. The goal of the game is to destroy the Sushi by either hitting them directly with a star or using the star to destroy other objects so that the sushi fall and hit the ground.

The game was made using the Unity Game Engine, and all art, assets and coding was created by myself.

Download instructions:
Please follow the link below to download the game from my GitHub Repo.
Click download option that appears on the page for build.zip file.
Once downloaded, unzip/extract the build from the folder and click on the "Angry birds assignment.exe" file to play the game.

Play Instructions are included in the game. Exit the game any time using Alt+F4.
