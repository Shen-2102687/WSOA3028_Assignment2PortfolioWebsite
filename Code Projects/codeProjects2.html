Calculate The Largest Sub-Square Matrix

This is a C++ programme that takes in any a square matrix of integers and calculates the greatest sub-square-matrix sum of the original matrix.

The programme runs on any IDE that can support C++ and has a console to run the code.

To use the programme, run the file on an IDE as described above and provide the input to the console as follows: The first line of input is an integer that dictates the dimensions of the square matrix, and the proceeding lines are the actual matrix.

For Example:
Input:
3
123
456
789

Output: 45
